Python Python diciptakan oleh Guido van Rossum pertama kali pada awal tahun 1990an. di Belanda Bahasa Python terinspirasi dari bahasa pemrograman ABC hingga saat ini, Guido masih tetap menjadi penulis utama untuk Python Meskipun begitu, Python bersifat open source, sehngga ribuan orang juga memiliki kontribusi dalam pengembangannya. Pada tahun 1995, Guido melanjutkan di Corporation for National Research Iniatiative (CNRI), Virginia Amerika, tempat dimana ia merilis beberapa versi dari Python.
